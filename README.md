## (IEEE Access'2024) BioBridge: Unified Bio Embeddings with Bridging Modality in Code-Switched EMR.

This repository contains the code and proposed BioBridge for our paper [BioBridge: Unified Bio-Embedding With Bridging Modality in Code-Switched EMR](https://arxiv.org/abs/2412.11671).
